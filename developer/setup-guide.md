# Setup Guide

## Prerequisites

- Unity 2023 LTS
- Git
- Visual Studio 2022 or VS Code

## Installation

1. Clone repository:
```bash
git clone https://github.com/squanchy667/TarotBattlegrounds-POC.git
```

2. Open in Unity Hub:
   - Add project from disk
   - Select `TarotBattlegrounds-POC/TarotBattlegrounds-POC`

3. Open scene:
   - Navigate to `Assets/Scenes/GameScene.unity`
   - Press Play
